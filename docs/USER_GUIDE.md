## 시작하기

### 시스템 접속
웹 브라우저에서 다음 주소로 접속합니다:

- Firefox 88 이상

## 로그인 방법

### 방법 1: 로컬 계정 로그인 (기본)

1. 로그인 페이지에서 **행원번호**와 **비밀번호**를 입력합니다
2. **로그인** 버튼을 클릭합니다

```
행원번호: user001
비밀번호: ****
[로그인]
```

**기본 제공 계정:**
- user001 / user002 / user003 / user004

### 방법 2: Swing SSO 로그인 (선택 사항)

> ⚠️ Swing SSO는 관리자가 활성화한 경우에만 사용 가능합니다.

#### 2-1. SSO 토큰 로그인
1. Swing 시스템에서 SSO 토큰을 발급받습니다
2. 로그인 페이지에서 **Swing SSO로 로그인** 버튼을 클릭합니다
3. SSO 토큰을 입력합니다

#### 2-2. 통합 ID/비밀번호 로그인
1. 로그인 페이지에서 **행원번호**와 **비밀번호**를 입력합니다
2. **Swing 인증** 옵션을 선택합니다
3. **로그인** 버튼을 클릭합니다

> 💡 Swing 인증을 사용하면 회사 통합 계정으로 로그인됩니다.

## 퀴즈 참여하기

### 1. 퀴즈 선택

로그인 후 진행 중인 퀴즈 목록이 표시됩니다:

```
📝 2025년 1월 내규 퀴즈
   기간: 2025-01-01 ~ 2025-12-31
   [퀴즈 시작하기]
```

**퀴즈 정보:**
- 총 5문제 출제
- 1회 참여당 약 5분 소요
- 월별 여러 번 참여 가능

### 2. 퀴즈 진행

#### 문제 유형

**1) O/X 문제**
```
Q1. 회사 노트북을 개인 용도로 사용해도 된다

○ O    ○ X

[제출]
```

**2) 빈칸 채우기**
```
Q2. 직원은 출근 후 [    ] 분 이내에 근태 시스템에 등록해야 한다

○ 10분  ○ 20분  ○ 30분  ○ 60분

[제출]
```

**3) 드래그 앤 드롭**
```
Q3. 개인정보 보호 관련 업무는 어느 부서의 책임인가?

[인사팀] [IT팀] [법무팀] [재무팀] [영업팀]

정답을 여기로 드래그 → [          ]

[제출]
```

**4) 타이핑**
```
Q4. 다음 문장을 정확히 입력하세요

"회사 기밀 정보는 외부에 유출할 수 없으며 위반 시 징계 대상이다"

[_____________________________________________]

[제출]
```

**5) 오류 찾기**
```
Q5. 다음 문장에서 맞춤법 오류를 찾으세요

직원은 [출근] 시 [회사] [보안카드]를 [착용]하고 [퇴근] 시에도
반드시 패용해야 한다

밑줄 친 단어 중 틀린 것을 클릭하세요

[제출]
```

#### 정답 제출

1. 답을 선택하거나 입력합니다
2. **제출** 버튼을 클릭합니다
3. 즉시 정답 여부를 확인할 수 있습니다

#### 오답 시
```
❌ 틀렸습니다

정답: 법무팀
설명: 개인정보 보호법에 따라 법무팀이 담당합니다.

[다시 풀기]
```

- 최대 2번까지 재도전 가능
- 2번 모두 틀리면 자동으로 다음 문제로 이동

#### 정답 시
```
✅ 정답입니다!

설명: 개인정보 보호법에 따라 법무팀이 담당합니다.

[다음 문제]
```

### 3. 럭키드로우 문제

**특별 조건 달성 시 출현:**
- 한 번에 맞춘 문제가 3개 이상일 때
- 바로 다음 문제가 **럭키드로우 문제**로 출제됩니다

```
🎰 Lucky Draw Question!

이 문제를 맞추면 추첨 기회가 주어집니다!

Q. 회사 창립일은 매년 공휴일로 지정된다

○ O    ○ X

[제출]
```

**럭키드로우 혜택:**
- 문제를 맞추면 자동으로 추첨 대상에 등록됩니다
- 월말에 추첨을 통해 경품을 받을 수 있습니다

### 4. 퀴즈 완료

5문제를 모두 풀면 결과 화면이 표시됩니다:

```
🎉 퀴즈 완료!

총점: 80점 (4/5 정답)
한 번에 맞춘 문제: 3개
럭키드로우 참여: ✓

📊 나의 순위: 15위 / 50명

[다시 도전하기]  [결과 보기]
```

## 럭키드로우

### 참여 조건
- 럭키드로우 문제를 맞춰야 합니다
- 한 번에 맞춘 문제가 3개 이상이면 럭키드로우 문제가 출제됩니다

### 당첨자 발표
- 매월 말일에 추첨 진행
- 당첨자에게 개별 통보
- 공지사항에 당첨자 명단 게시

### 경품 수령
1. 당첨 통보를 받으면 인사팀에 연락합니다
2. 신분 확인 후 경품을 수령합니다
3. 수령 확인서에 서명합니다

## 자주 묻는 질문

### Q1. 비밀번호를 잊어버렸어요
**A:** 관리자에게 문의하여 비밀번호를 재설정 받으세요.

### Q2. 몇 번까지 참여할 수 있나요?
**A:** 제한 없이 여러 번 참여할 수 있습니다. 최고 점수가 기록됩니다.

### Q3. 럭키드로우 문제가 안 나와요
**A:** 한 번에 맞춘 문제가 3개 이상이어야 럭키드로우 문제가 출제됩니다.

### Q4. 문제를 건너뛸 수 있나요?
**A:** 아니요, 순서대로 풀어야 합니다. 2번 틀리면 자동으로 넘어갑니다.

### Q5. 모바일에서도 되나요?
**A:** 네, 모바일 브라우저에서도 사용 가능합니다.

### Q6. Swing SSO 로그인이 안 돼요
**A:**
- Swing SSO가 활성화되어 있는지 확인하세요
- 올바른 계정 정보를 입력했는지 확인하세요
- 관리자에게 접근 권한이 있는지 문의하세요

### Q7. 답을 제출했는데 반응이 없어요
**A:**
- 인터넷 연결을 확인하세요
- 페이지를 새로고침해보세요
- 문제가 계속되면 관리자에게 문의하세요

### Q8. 점수가 이상해요
**A:**
- 한 번에 맞춘 문제: 20점
- 두 번 만에 맞춘 문제: 10점
- 틀린 문제: 0점

### Q9. 내 정보를 수정하고 싶어요
**A:** 관리자에게 문의하여 정보를 수정할 수 있습니다.

### Q10. 이전 퀴즈도 참여할 수 있나요?
**A:** 기간이 지난 퀴즈는 참여할 수 없습니다.

## 팁과 요령

### 높은 점수 받는 법
1. **내규를 숙지하세요**: 실제 내규를 읽어보면 도움이 됩니다
2. **차근차근 읽으세요**: 문제를 잘 읽고 답하세요
3. **타이핑 문제 주의**: 띄어�기와 문장부호까지 정확해야 합니다
4. **오류 찾기는 소거법**: 하나씩 확인해보세요

### 럭키드로우 참여하려면
1. **첫 시도에 맞추세요**: 한 번에 맞춰야 카운트됩니다
2. **집중하세요**: 실수하지 않도록 집중합니다
3. **기본 문제 먼저**: 쉬운 문제부터 한 번에 맞추세요

### 효율적인 학습
1. **복습하세요**: 틀린 문제의 설명을 잘 읽으세요
2. **반복 참여**: 여러 번 참여하면 문제 유형에 익숙해집니다
3. **동료와 공유**: 어려운 부분을 동료와 공유하세요
